🛡️ Prism Insurance Analytics Dashboard

🚀 A professional Power BI dashboard built to analyze insurance data, uncover claim trends, and support data-driven business decisions.

📌 Executive Summary

This project showcases the transformation of raw insurance data into meaningful insights using Power BI. It delivers a complete view of premiums, claims, coverage, and customer behavior, enabling better decision-making and performance tracking.

🎯 Business Problem

Insurance companies generate large datasets but struggle to answer key questions:

Which policies generate the most revenue?
Why are many claims rejected?
Which customer segments are high risk?
How effective is policy retention?

Lack of centralized analytics leads to delayed and inefficient decisions.

💡 Solution

An interactive dashboard was developed to:

Track Premium, Coverage, and Claim metrics
Analyze claim status (Rejected, Settled, Pending)
Identify top-performing policy types
Monitor active vs inactive policies
Evaluate claims by age group
Enable filtering by Policy, Claim, and Customer ID
